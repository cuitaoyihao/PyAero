Welcome to PyAERO's documentation!
==================================

PyAero is an airfoil contour analysis and CFD meshing tool written in Python.
PyAero is open-source.

The code ist hosted on GitHub: `PyAero source code <https://github.com/chiefenne/PyAero>`_
